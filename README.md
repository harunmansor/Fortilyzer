# Fortilyzer
A script to quickly perform an analysis of a Fortigate firewall log

## Installation
```bash
git clone https://github.com/Cresp0x/Fortilyzer.git
cd Fortilyzer
chmod +x fortilyzer.sh
```

## Usage 
```
./fortilyzer.sh /path/to/logfile.log
```
